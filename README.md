## 🚀 خطوات الرفع

### الخطوة 1: الحصول على توكن البوت

1. افتح تيليغرام وابحث عن **@BotFather**
2. أرسل له `/newbot`
3. اختر اسم البوت: **بوت الأذكار**
4. **احفظ التوكن**

---

### الخطوة 2: نشر البوت على Render

1. سجّل دخول على [render.com](https://render.com)
2. اضغط **New → Background Worker**
3. اربطه بـ GitHub repository
4. اختر الـ Branch: `main`

**إعدادات:**
```
Build Command:   pip install -r requirements.txt
Start Command:   python main.py
```

5. اضغط **Add Environment Variable:**
   - Key: `BOT_TOKEN`
   - Value: التوكن من BotFather

6. اضغط **Create Background Worker**
